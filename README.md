# Pacote Office — Landing Page

Landing page de oferta para o Pacote Office.

## Estrutura

```
├── index.html   # Página principal
├── style.css    # Todos os estilos
├── script.js    # Contador, FAQ accordion, animações
└── README.md
```

## Como hospedar no Vercel

1. Suba esta pasta para um repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e faça login com sua conta GitHub
3. Clique em **"Add New Project"**
4. Selecione o repositório
5. Clique em **"Deploy"** — pronto!

O Vercel detecta automaticamente que é um site estático (HTML/CSS/JS) e publica em segundos.
